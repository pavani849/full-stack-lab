Understanding Different Types of Lists in HTML

1. Ordered List

Below is an ordered list, which uses numbers to order the items:

  1. First item
  2. Second item
  3. Third item

2. Unordered List

Below is an unordered list, which uses bullets for each item:

  • Apple
  • Banana
  • Orange

3. Nested Unordered List

This is an unordered list with nested unordered items:

  • Fruits
    • Apple
    • Banana
    • Grapes
  • Vegetables
    • Carrot
    • Potato
    • Spinach

5. Definition List

This is an example of a definition list, where each term has a definition:

HTML
A standard markup language used to create web pages.
CSS
A stylesheet language used to describe the presentation of a document written in HTML.
JavaScript
A programming language used to create dynamic and interactive effects on web pages.
